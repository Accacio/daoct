#+title: Daoct

* About
Command-line tool to generate DAOCT (Deterministic
Automaton With Outputs And Conditional Transitions) using data from a
log in csv format with inputs and outputs.

* Dependencies
- matplotlib
- tikzplotlib
- scipy

* Usage
#+begin_src sh
./daoct -h
#+end_src
